## мӢӨмҠө нҷҳкІҪ мӢңмһ‘

```bash
# нҳёмҠӨнҠём—җм„ң мӢӨн–ү
# 1. м ҖмһҘмҶҢ нҒҙлЎ  (мөңмҙҲ 1нҡҢ)
git clone https://github.com/fankh/elk-siem-lab.git
cd elk-siem-lab

# 2. ELK Stack м „мІҙ мӢңмһ‘ (ES, Kibana, Logstash, Filebeat, tools)
docker-compose up -d

# 3. м»Ён…Ңмқҙл„Ҳ мғҒнғң нҷ•мқё
docker-compose ps
# кё°лҢҖ: elasticsearch, kibana, logstash, filebeat, tools лӘЁл‘җ Up

# 4. ES мғҒнғң нҷ•мқё (~60мҙҲ лҢҖкё°)
curl http://localhost:9200/_cluster/health?pretty
# кё°лҢҖ: "status" : "yellow"
```

> **м°ёкі **: мІ« мӢӨн–ү мӢң Docker мқҙлҜём§Җ лӢӨмҡҙлЎңл“ңм—җ ~5л¶„, ES мҙҲкё°нҷ”м—җ ~60мҙҲ, Logstash лЎңк·ё мІҳлҰ¬м—җ ~90мҙҲ мҶҢмҡ”

> **CLI мӢӨн–ү нҷҳкІҪ**
> | нҷҳкІҪ | лӘ…л № |
> |------|------|
> | **tools м»Ён…Ңмқҙл„Ҳ (к¶ҢмһҘ)** | `docker exec -it tools bash` вҶ’ лӮҙл¶Җм—җм„ң curl, python3, sigma мӮ¬мҡ© |
> | **macOS/Linux** | нҳёмҠӨнҠё н„°лҜёл„җм—җм„ң `curl localhost:9200/...` |
> | **Windows PowerShell** | `curl.exe localhost:9200/...` лҳҗлҠ” лёҢлқјмҡ°м Җ м§Ғм ‘ м ‘мҶҚ |
> | **лёҢлқјмҡ°м Җ** | http://localhost:9200 (ES) / http://localhost:5601 (Kibana) |

## м»ҙнҸ¬л„ҢнҠё м„Өм№ҳ: Filebeat & Logstash

### Filebeat м„Өм№ҳ

FilebeatлҠ” кІҪлҹү лЎңк·ё мҲҳм§‘ м—җмқҙм „нҠёмһ…лӢҲлӢӨ. ліё мӢӨмҠөм—җм„ңлҠ” DockerлЎң мӢӨн–үлҗ©лӢҲлӢӨ.

**Docker (ліё мӢӨмҠө)**
```bash
# docker-compose.ymlм—җ нҸ¬н•ЁлҗЁ (мһҗлҸҷ мӢӨн–ү)
docker-compose up -d filebeat
```

**мҲҳлҸҷ м„Өм№ҳ (м°ёкі мҡ©)**
```bash
# Linux
curl -L -O https://artifacts.elastic.co/downloads/beats/filebeat/filebeat-8.12.0-linux-x86_64.tar.gz
tar xzvf filebeat-8.12.0-linux-x86_64.tar.gz

# macOS
brew install elastic/tap/filebeat-full

# Windows - elastic.coм—җм„ң ZIP лӢӨмҡҙлЎңл“ң
```

**н•өмӢ¬ м„Өм • (filebeat/filebeat.yml)**
```yaml
filebeat.inputs:
  - type: log
    paths: ["/var/log/sample/web-access.log"]
    tags: ["web", "access"]          # нғңк·ёлЎң Logstashм—җм„ң лқјмҡ°нҢ…
  - type: log
    paths: ["/var/log/sample/sysmon.json"]
    tags: ["sysmon", "endpoint"]
    json.keys_under_root: true       # JSON н•„л“ңлҘј мөңмғҒмң„лЎң
  - type: log
    paths: ["/var/log/sample/suricata-eve.json"]
    tags: ["suricata", "ids"]
    json.keys_under_root: true

output.logstash:
  hosts: ["logstash:5044"]           # Logstash нҸ¬нҠё 5044лЎң м „мҶЎ
```

| м„Өм • | м„ӨлӘ… |
|------|------|
| `type: log` | нҢҢмқј кё°л°ҳ лЎңк·ё мҲҳм§‘ |
| `tags` | Logstashм—җм„ң мқҙлІӨнҠё лқјмҡ°нҢ…м—җ мӮ¬мҡ© |
| `json.keys_under_root` | JSON лЎңк·ёмқҳ н•„л“ңлҘј мөңмғҒмң„лЎң м¶”м¶ң |
| `output.logstash` | Logstash Beats мһ…л Ҙ(5044)мңјлЎң м „мҶЎ |

### Logstash м„Өм№ҳ

LogstashлҠ” лЎңк·ё нҢҢмӢұ, ліҖнҷҳ, лқјмҡ°нҢ…мқ„ мҲҳн–үн•ҳлҠ” нҢҢмқҙн”„лқјмқё м—”м§„мһ…лӢҲлӢӨ.

**Docker (ліё мӢӨмҠө)**
```bash
docker-compose up -d logstash
```

**мҲҳлҸҷ м„Өм№ҳ (м°ёкі мҡ©)**
```bash
# Linux
wget https://artifacts.elastic.co/downloads/logstash/logstash-8.12.0-linux-x86_64.tar.gz
tar xzvf logstash-8.12.0-linux-x86_64.tar.gz

# macOS
brew install elastic/tap/logstash-full
```

**нҢҢмқҙн”„лқјмқё кө¬мЎ° (Input вҶ’ Filter вҶ’ Output)**
```
logstash/pipeline/
в”ңв”Җв”Җ web-access.conf     # мӣ№ лЎңк·ё: Grok нҢҢмӢұ + GeoIP + кіөкІ© нғңк№…
в”ңв”Җв”Җ sysmon.conf         # Sysmon: ECS л§Өн•‘ + мқҳмӢ¬ н”„лЎңм„ёмҠӨ нғңк№…
в””в”Җв”Җ suricata.conf       # Suricata: IDS м•ҢлҰј нҢҢмӢұ + кіөкІ© л¶„лҘҳ
```

| нҢҢмқҙн”„лқјмқё | Input | Filter | Output Index |
|-----------|-------|--------|-------------|
| web-access.conf | file (web-access.log) | grok, date, geoip, useragent, urldecode | security-web-* |
| sysmon.conf | file (sysmon.json) | date, mutate (ECS rename) | security-sysmon-* |
| suricata.conf | file (suricata-eve.json) | date, mutate (ECS rename), geoip | security-suricata-* |

**нғңк·ё кё°л°ҳ лқјмҡ°нҢ…**: к°Ғ conf нҢҢмқјмқҖ `if "web" in [tags]` мЎ°кұҙмңјлЎң н•ҙлӢ№ лЎңк·ёл§Ң мІҳлҰ¬н•©лӢҲлӢӨ.

### мҷң Filebeat вҶ’ Logstash вҶ’ ES кө¬мЎ°мқёк°Җ?

Filebeatм—җм„ң ESлЎң м§Ғм ‘ м „мҶЎн•  мҲҳлҸ„ мһҲм§Җл§Ң, ліё мӢӨмҠөм—җм„ңлҠ” LogstashлҘј кІҪмң н•©лӢҲлӢӨ.

| н•ӯлӘ© | Filebeat вҶ’ ES (м§Ғм ‘) | Filebeat вҶ’ Logstash вҶ’ ES |
|------|:-------------------:|:------------------------:|
| **нҢҢмӢұ** | Ingest Pipelineл§Ң (м ңн•ңм Ғ) | Grok, JSON, CSV л“ұ н’Қл¶Җн•ң н•„н„° |
| **ліҖнҷҳ/ліҙк°•** | кё°ліё н”„лЎңм„ём„ңл§Ң | GeoIP, UserAgent, urldecode |
| **кіөкІ© нғңк№…** | л¶Ҳк°Җ | regex мЎ°кұҙл¶Җ нғңк№… (sqli, xss л“ұ) |
| **лқјмҡ°нҢ…** | 1 input вҶ’ 1 index | нғңк·ё кё°л°ҳ лӢӨмӨ‘ мқёлҚұмҠӨ л¶„кё° |
| **лІ„нҚјл§Ғ** | м—ҶмқҢ (ES мһҘм•  мӢң мң мӢӨ) | Persistent QueueлЎң мң мӢӨ л°©м§Җ |
| **ES л¶Җн•ҳ** | нҢҢмӢұмқҙ ESм—җ м§‘мӨ‘ | Logstashк°Җ нҢҢмӢұ, ESлҠ” м ҖмһҘл§Ң |

**ліё мӢӨмҠөм—җм„ң Logstashк°Җ н•„мҲҳмқё мқҙмң :**
- GrokмңјлЎң Apache Combined лЎңк·ёлҘј н•„л“ң лӢЁмң„лЎң нҢҢмӢұ
- `urldecode` нӣ„ regexлЎң SQLi/XSS/Path Traversal **кіөкІ© нғңк№…**
- GeoIPлЎң мҷёл¶Җ IP вҶ’ көӯк°Җ/лҸ„мӢң ліҙк°•
- нғңк·ё кё°л°ҳмңјлЎң `security-web-*` / `security-sysmon-*` / `security-suricata-*` **мқёлҚұмҠӨ л¶„кё°**

**мӢӨл¬ҙ м•„нӮӨн…ҚмІҳ л№„көҗ:**

```
мҶҢк·ңлӘЁ (нҢҢмӢұ л¶Ҳн•„мҡ”):  Filebeat вҶ’ ES м§Ғм ‘
мӨ‘к·ңлӘЁ (нҢҢмӢұ н•„мҡ”):    Filebeat вҶ’ Logstash вҶ’ ES  вҶҗ ліё мӢӨмҠө
лҢҖк·ңлӘЁ (лІ„нҚј н•„мҡ”):    Filebeat вҶ’ Kafka вҶ’ Logstash вҶ’ ES
```

### 2-5. web-access.conf н•„н„° мІҙмқё мғҒм„ё

мӣ№ лЎңк·ёмқҳ м „мІҙ мІҳлҰ¬ нқҗлҰ„мһ…лӢҲлӢӨ:

```ruby
filter {
  if "web" in [tags] {
    # 1лӢЁкі„: Grok нҢҢмӢұ (мӣҗліё лЎңк·ё вҶ’ кө¬мЎ°нҷ” н•„л“ң)
    grok {
      match => {
        "message" => '%{IPORHOST:source.ip} - %{DATA:user.name} \[%{HTTPDATE:timestamp}\] "%{WORD:http.request.method} %{URIPATHPARAM:url.path} HTTP/%{NUMBER}" %{NUMBER:http.response.status_code:int} %{NUMBER:http.response.body.bytes:int} "%{DATA:http.request.referrer}" "%{DATA:user_agent.original}"'
      }
    }

    # 2лӢЁкі„: нғҖмһ„мҠӨнғ¬н”„ ліҖнҷҳ
    date { match => ["timestamp", "dd/MMM/yyyy:HH:mm:ss Z"] }

    # 3лӢЁкі„: GeoIP ліҙк°• (мҷёл¶Җ IP вҶ’ көӯк°Җ/лҸ„мӢң)
    geoip { source => "source.ip" target => "source.geo" }

    # 4лӢЁкі„: User-Agent нҢҢмӢұ
    useragent { source => "user_agent.original" target => "user_agent" }

    # 5лӢЁкі„: URL л””мҪ”л”© (кіөкІ© нғҗм§ҖлҘј мң„н•ҙ)
    mutate { copy => { "url.path" => "url.path_decoded" } }
    urldecode { field => "url.path_decoded" }

    # 6лӢЁкі„: кіөкІ© нҢЁн„ҙ л§Өм№ӯ вҶ’ нғңк·ё м¶”к°Җ
    if [url.path_decoded] =~ /(?i)(union.+select|or.+1.+=.+1|drop\s+table|sleep\(|'--)/
    { mutate { add_tag => ["attack", "sqli"] } }

    if [url.path_decoded] =~ /(?i)(<script|javascript:|onerror\s*=)/
    { mutate { add_tag => ["attack", "xss"] } }

    if [url.path_decoded] =~ /(\.\.\/|\.\.%2[fF])/
    { mutate { add_tag => ["attack", "path_traversal"] } }
  }
}
```

**мІҳлҰ¬ нқҗлҰ„ мҡ”м•Ҫ:**
